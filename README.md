# SEO Checker

Проверка доменов по SEO-критериям. Результаты в CSV.

## Установка и запуск (Windows)

### Шаг 1. Установите Python
Скачайте и установите Python 3.10.12+
https://www.python.org/downloads/

### Шаг 2. Создайте виртуальное окружение
```cmd
python -m venv .venv
.venv\Scripts\activate
```

### Шаг 3. Установите зависимости
```cmd
pip install -r requirements.txt
```

### Шаг 4. Запустите приложение
```cmd
python app.py
```

### Шаг 5. Откройте в браузере
```
http://127.0.0.1:5000
```

## Что проверяет
- Коды ответов (40x/50x)
- Редиректы
- Noindex/Nofollow, Canonical
- Title, Description
- Sitemap.xml, Robots.txt
- Страница 404
- H1
- Alt для изображений
- Доменные дубли (http/https, www)
- HTML структура (H1–H6, P)
- Дубли заголовков (H1/H2/H3)

## Использование
1. Вставьте домены в поле (по одному в строке)
2. Выберите проверки (по умолчанию все включены)
3. Нажмите **Старт**
4. После завершения нажмите **Скачать CSV**
